#### 说明

一个 有道翻译和百度翻译的nodejs 小工具 只支持 中文/英文 英文/中文 翻译

使用如下：

        Usage: translator [options]


        Commands:

        youdao|y            有道翻译
        baidu|b [options]   百度翻译

        Options:

        -h, --help     output usage information
        -V, --version  output the version number

        Example:

        $ translator y
        $ translator b -c|-e
